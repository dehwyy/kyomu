# Commit naming

## General
- {`TYPE`} [{`FILES`/`FOLDERS`} or {`WHAT_WAS_DONE`}]

### Types
- `DOC`  - documentation.
- `FIX`  - fix unwanted behavior somewhere.
- `FEAT` - new feature.
- `REF`  - code refactor (reorganized files, folders structures, etc.)
- `OPT`  - code optimization (code power up!)
